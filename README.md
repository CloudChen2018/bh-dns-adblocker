# BlackHand DNS ADBlocker

使用Docker架設阻擋廣告的DNS伺服器

Feature
=========================
- 使用Docker架設可以阻擋廣告的DNS
- 每天自動更新黑名單

Installation
=========================
- [安裝Docker](https://docs.docker.com/install/linux/docker-ce/debian/)
- clone本項目以後，在根目錄下指令 `docker-compose up -d`
